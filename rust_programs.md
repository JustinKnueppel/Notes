## `zoxide`

`cargo install zoxide`

Add `eval "$(zoxide init zsh)"` to `~/.profile`

## `starship`

Requires openssl development packages

`sudo zypper install libopenssl-devel`

Install with cargo

`cargo install starship`

Add line to `.profile`

`eval "$(starship init zsh)"
